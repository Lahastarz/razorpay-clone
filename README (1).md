## ✨ Features

| Feature | Description |
|---|---|
| 🏠 **Home Page** | Razorpay-inspired landing page with modern UI |
| 🔐 **Authentication** | Secure Login & Signup system |
| 📊 **Dashboard** | Interactive dashboard with banking & payment overview |

## 🚀 Getting Started

### Prerequisites
- ✅ Java JDK 8 or above
- ✅ Apache Tomcat (v9 or above)
- ✅ Eclipse IDE or IntelliJ IDEA

### Steps

**1. Clone the repository**
```bash
git clone https://github.com/YOUR_USERNAME/razorpay-clone.git
```

**2. Import into your IDE**
- Open **Eclipse** → Import → Dynamic Web Project
- Or open **IntelliJ** → Open as Web/Maven Project

**3. Deploy on Tomcat**
- Add project to your Tomcat server
- Start the server ▶️

**4. Open in browser**
```
http://localhost:8080/razorpay-clone/HomePage.jsp
```
