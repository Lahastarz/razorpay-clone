# ⚙️ Requirements

Everything you need to run the **Razorpay Clone** project on your local machine.

---

## ☕ Java

| Requirement | Version |
|---|---|
| Java JDK | 8 or above |
| Java EE (Enterprise Edition) | Supported via Tomcat |

👉 Download: [Java JDK](https://www.oracle.com/java/technologies/downloads/)

---

## 🐱 Apache Tomcat (Web Server)

| Requirement | Version |
|---|---|
| Apache Tomcat | v9.0 or above |

👉 Download: [Apache Tomcat](https://tomcat.apache.org/download-90.cgi)

---

## 💻 IDE (Code Editor)

Any one of the following:

| IDE | Recommended For |
|---|---|
| Eclipse IDE for Enterprise Java | ⭐ Best for JSP projects |
| IntelliJ IDEA (Ultimate) | Great alternative |
| VS Code (with Java extensions) | Lightweight option |

👉 Download Eclipse: [eclipse.org](https://www.eclipse.org/downloads/)

---

## 🌐 Browser

Any modern browser works:

| Browser | Status |
|---|---|
| Google Chrome | ✅ Recommended |
| Mozilla Firefox | ✅ Supported |
| Microsoft Edge | ✅ Supported |
| Safari | ✅ Supported |

---

## 📦 No Extra Libraries Needed!

This project uses **pure JSP, HTML, CSS & JavaScript** — no external libraries or package managers (like npm or Maven) are required. Just set up Tomcat and you're good to go! ✅

---

## 🧾 Quick Checklist

Before running the project, make sure you have:

- [ ] Java JDK 8+ installed
- [ ] Apache Tomcat 9+ installed
- [ ] Eclipse or IntelliJ IDE set up
- [ ] Project imported as a Dynamic Web Project
- [ ] Tomcat configured in your IDE
- [ ] A modern browser to view the output
